# Icons

These icons are required by `tauri.conf.json`. Replace before shipping.

Generate from a source PNG (1024×1024, transparent background) with:

```
npx @tauri-apps/cli icon path/to/source.png
```

Tauri will write `32x32.png`, `128x128.png`, `128x128@2x.png`, `icon.icns`,
and `icon.ico` into this directory.
